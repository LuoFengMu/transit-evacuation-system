# 数据需求与来源文档

## v0.4.0 公交-轨道协同疏散仿真系统

---

## 一、数据分类总览

| 类别 | 真实/生成 | 来源 |
|------|----------|------|
| 道路路网 | 真实 | OSM → OSMnx |
| SUMO 仿真路网 | 真实 | OSM → netconvert |
| 公交站点 | 真实 | OSM → Overpass API |
| 轨道站点 | 真实坐标 + 估算容量 | OSM + 公开资料 |
| 轨道线路 | 推算 | 公开报道 (6B编组/发车间隔) |
| 公交场站 | 真实 | OSM → Overpass API |
| 安全区域 | 真实坐标 + 估算容量 | OSM → OSMnx |
| 疏散需求点 | 生成 | 随机生成 + 量级缩放 |
| 公交车辆/调度 | 生成 | OR-Tools CVRP |
| SUMO 轨迹 | 生成 | SUMO 仿真输出 |

## 四、开源数据获取方式一览

| 数据 | 工具/API | 命令示例 |
|------|---------|---------|
| 道路路网 | OSMnx | `ox.graph_from_place("Xuzhou, Jiangsu, China")` |
| 公交站 | Overpass API | `node[highway=bus_stop](bbox)` |
| 公交场站 | Overpass API | `node[amenity=bus_station](bbox)` |
| 安全区域 | OSMnx | `ox.features_from_place(tags={"leisure":"park"})` |
| 人口分布 | WorldPop | `worldpop.org → China 100m population` |
| 步行网络 | OSMnx | `ox.graph_from_place(network_type="walk")` |

## 五、总结

**地理骨架 (路网/公交站/轨道站/场站) 来自 OpenStreetMap 开源数据。运营参数 (容量/发车间隔/步行速度) 按规范估算。人流数据 (需求点/调度方案/仿真结果) 由模型生成。**

**论文中需明确标注: ☆ 字段为真实数据支撑的最小需求；★ 字段为提升精度的扩展需求，当前版本部分使用估算值替代。**
