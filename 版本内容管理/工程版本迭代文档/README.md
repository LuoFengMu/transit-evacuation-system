# 工程版本迭代文档索引

本目录按照系统工程开发的版本更迭进行文档划分。每个版本文档都可以作为对应阶段的开发说明、验收清单和 Release Note 基础材料。

## 文档体系

| 文档 | 内容 |
| --- | --- |
| `../工程实施与版本管理方案.md` | **权威主文档**：技术选型、架构、数据规范、版本管理方案 |
| `../拆分文档/01_版本路线详解.md` | 每版本详细模块清单、接口定义与依赖关系 |
| `../拆分文档/02_场景实验操作手册.md` | 如何创建场景、运行实验、管理数据版本、对比方案 |
| `../拆分文档/03_发布与运维指南.md` | 发布 checklist、回滚操作、兼容性矩阵、日常运维 |
| `00_工程版本迭代总览.md` | 本文件：版本路线总览和开发节奏 |
| `v0.1.0_最小可运行原型.md` | 路网、事件、需求点、避难点、静态最短路径 + Streamlit 地图 |
| `v0.2.0_公交调度优化版本.md` | 公交车辆、depot、OR-Tools 调度 + 简化仿真验证 |
| `v0.3.0_SUMO动态仿真版本.md` | SUMO 路网、route 文件、TraCI 动态仿真 |
| `v0.4.0_公交轨道协同版本.md` | 轨道站点容量、线路运能、接驳分流和站点压力 |
| `v0.5.0_Web可视化版本.md` | v0.5.0 可复现实验平台（CLI、批量实验、manifest、summary.csv；文件名沿用历史命名） |
| `v0.6.0_多事件中心大客流疏散.md` | 多事件 YAML、需求合并、全局调度、分事件指标 |
| `v1.0.0_论文与答辩稳定版本.md` | 稳定发布、实验固化、环境固化和论文交付 |

## 版本推进顺序

```text
v0.1.0 → v0.2.0 → v0.3.0 → v0.4.0 → v0.5.0 → v0.6.0 → v1.0.0
```

## 每完成一个版本，同步更新

- 对应版本文档（本目录）
- `CHANGELOG.md`
- `VERSION`
- 场景配置（`configs/scenarios/`）
- 数据清单（`data/manifest/`）
- 实验报告（`experiments/`）
- Release Note（`docs/release_notes/`）
- `simulation_run` 关联表记录
